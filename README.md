# 김비서 Agent - 일일 업무 자동 브리핑 시스템

## 📌 개요

**김비서 Agent**는 일일 업무 현황을 자동으로 분석하고 전문적인 경영진 브리핑을 생성하는 지능형 에이전트입니다. 5개의 데이터 파일(매출, 업무, 일정, 프로젝트, 회의)을 실시간으로 분석하여 마크다운 형식의 상세한 보고서를 제공합니다.

**특징:**
- ✅ 완전 자동화된 일일 브리핑 생성
- ✅ 5개 데이터 파일 통합 분석
- ✅ 긴급 업무 자동 식별
- ✅ 프로젝트 위험도 평가
- ✅ 매출 통계 및 트렌드 분석
- ✅ 전문적인 마크다운 형식 출력

## 🎯 주요 기능

### 1. 모든 데이터 파일 읽기
- **매출데이터.csv**: 전자기기, 생활용품 판매 현황
- **업무목록.csv**: 마케팅, 콘텐츠, 리서치, 운영, 보고 업무
- **주간일정.txt**: 주간 회의 및 미팅 일정
- **프로젝트현황.csv**: 진행 중인 6개 프로젝트
- **회의록.txt**: 최근 회의 결정사항 및 실행 과제

### 2. 오늘의 업무와 일정 추출
```markdown
## 🔴 긴급 할 일 (오늘)
- 워크숍 후기 작성 (D-2, 마감: 4월 11일)
- 진행 중인 프로젝트 3개 모니터링

## 📋 예약된 회의
- (데이터 기준 주간일정이 3월 기반이므로 업데이트 필요)
```

### 3. 최신 업무 현황 정리
| 상태 | 업무 수 | 설명 |
|------|--------|------|
| 🟢 완료 | 1건 | SNS 콘텐츠 업로드 |
| 🟠 진행 중 | 3건 | 프로모션, 영상 제작, 설문 분석 |
| 🟡 대기 | 7건 | 우선순위별 순서 대기 |

### 4. 매출 요약
- **기간**: 2026년 1월 5일 ~ 2월 10일
- **상위 제품**: 무선 이어폰, 블루투스 스피커, 텀블러
- **주요 지역**: 서울, 부산, 대구
- **분석**: B안(영상형) 광고 클릭률 2.3배 높음

### 5. 일일 브리핑 제공
```markdown
# 김비서 Agent - 일일 브리핑

## 📋 오늘의 일정 및 업무
## 📊 매출 현황
## 🎯 프로젝트 현황
## 📝 진행 중인 업무
## 🎤 회의 내용
## 📅 내일(4월 10일) 예상
## 🚨 주의사항
## 💡 권고사항
```

## ⚙️ 설정 (settings.json)

### 에이전트 설정
```json
{
  "agents": {
    "김비서": {
      "name": "Kim Secretary Agent",
      "enabled": true,
      "config": {
        "dataFolder": "./김비서-데이터",
        "updateFrequency": "daily",
        "briefingTime": "09:00"
      }
    }
  }
}
```

### 브리핑 설정
```json
{
  "briefingConfig": {
    "schedule": "daily",
    "time": "09:00",
    "timezone": "Asia/Seoul",
    "outputFormat": "markdown",
    "outputPath": "./김비서-데이터/브리핑_{date}.md"
  }
}
```

```json
{
  "analysisRules": {
```

## 📈 분석 기준

### 업무 우선순위 분류
```
🔴 긴급 (D-1, D-0)     → 오늘 또는 내일 마감
🟠 높음                → 3일 이내 마감
🟡 보통                → 1주 이내 마감
🟢 낮음                → 1주 이상 또는 완료
```

### 프로젝트 상태 평가
```
준비중 (0-10%)    → 초기 단계, 계획 수립
기획중 (10-20%)   → 상세 계획 작성
진행중 (20-80%)   → 활발한 실행
마무리 (80-95%)   → 최종 조정 및 검수
완료 (95-100%)    → 완성 및 배포
```

## 🐛 트러블슈팅

### 문제: 파일을 찾을 수 없음
**해결**: `settings.json`의 `dataPath` 확인
```json
"dataPath": "./김비서-데이터"
```

### 문제: 날짜가 잘못 해석됨
**해결**: 모든 파일의 날짜 형식 통일 (YYYY-MM-DD)
```bash
# 확인
grep -n "날짜" 김비서-데이터/매출데이터.csv
```

### 문제: CSV 파싱 에러
**해결**: 인코딩 확인 (UTF-8) 및 구분자 확인 (,)
```bash
file -I 김비서-데이터/*.csv
```

### 문제: 브리핑이 생성되지 않음
**해결**: 
1. 출력 폴더 존재 확인
2. 쓰기 권한 확인
3. 디스크 공간 확인

```bash
# 권한 확인 및 변경
ls -ld 김비서-데이터/
chmod 755 김비서-데이터/
```

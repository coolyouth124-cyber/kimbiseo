# 🤖 김비서 (Kim Secretary)

> 마케팅팀을 위한 AI 기반 데이터 분석 및 업무 관리 플랫폼

**프로젝트 시작**: 2026년 3월 10일  
**최종 업데이트**: 2026년 4월 16일  
**상태**: ✅ 운영 중

---

## 📋 프로젝트 개요

김비서는 마케팅팀의 **일일 업무 자동화**, **데이터 분석**, **성과 추적**을 위한 통합 플랫폼입니다.

### 🎯 핵심 가치제안
- 📊 **실시간 대시보드**: 모든 데이터를 한눈에
- 🤖 **AI 브리핑**: `/김비서` 한 명령어로 일일 업무 정리
- 📈 **스마트 분석**: 매출, 프로젝트, 업무 현황 자동 분석
- 🎨 **프리미엄 UI**: 글래스모피즘, 다크/라이트 모드

---

## 🚀 주요 기능

### 1️⃣ **통합 대시보드** (`dashboard.html`)
- 📌 할 일 목록 (우선순위/상태 색상화)
- 📅 주간 일정 표
- 🎯 프로젝트 진행률 (프로그레스 바)
- 💰 매출 요약 (통계 카드)
- 🔥 오늘의 핫한 뉴스 (자동 수집)
- 🌓 다크/라이트 모드 자동 전환

### 2️⃣ **매출 분석 차트** (`chart.html`)
- 📈 월별 매출 추이 (선 그래프)
- 📊 제품별 비교 (막대 그래프)
- 💹 실시간 통계 계산
- 🎨 파란/초록 계열 색상
- 📋 상세 데이터 테이블

### 3️⃣ **회의록 보고서** (`meeting-result.html`)
- 📝 회의 기본 정보
- 💬 논의 내용 요약
- ✅ 액션 아이템 추적
- 📋 마감일 관리
- 🖨️ 인쇄 최적화

### 4️⃣ **웹사이트 분석** (`report.html`)
- 🌐 사이트 구조 분석
- 🎨 디자인 및 색상 평가
- ✨ 주요 기능 정리
- 👍 잘한 점 (Strengths)
- 🎯 개선점 (Opportunities)

### 5️⃣ **프로세스 다이어그램** (`diagram.svg`)
- 🔄 5단계 업무 흐름 시각화
- 📌 기획 → 제작 → 검토 → 배포 → 분석
- 🎨 파스텔 색상 (각 단계별)
- 💫 피드백 루프 포함

### 6️⃣ **김비서 AI 에이전트** (`.kim-secretary.sh` / `kim_secretary_agent.py`)
- 🤖 `/김비서` 슬래시 명령어
- 📊 전체 데이터 자동 분석
- 💡 오늘의 업무 브리핑
- 🎯 우선순위별 추천
- 📅 마감 임박 업무 알림

### 7️⃣ **통합 네비게이션**
- 📊 대시보드
- 📋 회의록
- 📈 매출 현황
- 📌 업무 프로세스
- 📝 사이트 분석

## 🛠️ 사용 기술

### **Frontend**
- **HTML5** - 시맨틱 마크업
- **CSS3** - Flexbox, Grid, 글래스모피즘
- **JavaScript (Vanilla)** - 프레임워크 없이 순수 JS
- **Canvas API** - 차트 그리기 (외부 라이브러리 없음)
- **localStorage** - 테마 설정 저장

### **Design**
- 🎨 **글래스모피즘** - 반투명 카드, 블러 효과
- 🌈 **그라디언트 배경** - 애니메이션 배경
- 🌓 **다크/라이트 모드** - CSS 변수 기반
- 📱 **반응형 디자인** - 모바일~데스크톱

### **데이터 처리**
- 📊 **CSV 파싱** - 순수 JavaScript
- 📈 **차트 생성** - HTML5 Canvas
- 🔢 **수치 포맷팅** - 천 단위 쉼표, 통화 변환

### **AI & 자동화**
- 🤖 **Claude API** - 데이터 분석 및 브리핑
- 🔄 **Bash 스크립트** - 자동화 작업
- 🐍 **Python** - AI 에이전트 구현
- 📅 **Cron** - 정기 실행 (선택사항)

### **개발 도구**
- 🔧 **Git** - 버전 관리
- 📝 **Markdown** - 문서화
- 🚀 **GitHub** - 원격 저장소

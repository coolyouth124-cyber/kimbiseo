## 📁 데이터 소스

김비서는 다음 폴더의 데이터를 자동으로 읽습니다:

```

├── 주간일정.txt             (이번 주 일정)
├── 프로젝트현황.csv         (프로젝트 진행률)
└── 회의록.txt               (최신 회의 내용)
```

## 🔄 자동 업데이트 기능

### 설정 방법

`.claude/launch.json`에 다음을 추가하면 매일 자동으로 브리핑을 받을 수 있습니다:

```json
{
  "version": "0.0.1",
  "configurations": [
    {
      "name": "김비서 일일 브리핑",
      "runtimeExecutable": "bash",
      "runtimeArgs": ["/Users/joyunchae/Desktop/푸쉬업/워크숍-실습/.kim-secretary.sh"],
      "schedule": "0 9 * * *",
      "enabled": true
    }
  ]
}
```

**설정 후**: 매일 오전 9:00에 자동으로 브리핑이 생성됩니다.

## 🆘 자주 묻는 질문

### Q: 데이터가 자동으로 업데이트되나요?
**A:** 아니오. CSV 파일을 수정하면 다음 브리핑부터 반영됩니다. (대시보드는 실시간 업데이트됨)

### Q: 특정 날짜의 브리핑을 보려면?
**A:** 나중에 날짜 인자 기능을 추가할 예정입니다.

### Q: 브리핑을 파일로 저장할 수 있나요?
**A:** 터미널에서 `bash .kim-secretary.sh > 브리핑_$(date +%Y%m%d).txt`로 저장 가능합니다.
